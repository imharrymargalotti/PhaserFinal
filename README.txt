Description:

Welcome to FEEL THE BERN! In this game you will play as Bern Sandwiches. The object of the game is to infiltrate the Drumpf House where you will eventually face the evil dictator Drumpf himself. In order to defeat him, you must use your trusty squirt gun to eliminate his terrible spray tan. GOOD LUCK

_______________________
Controls:

Move Left: left key
Move Right: right key
Jump: up key
Shoot: SpaceBar

_______________________
About:

This game was developed with the JavaScript library phaser. It utilizes basic physics, shooting mechanics, and artificial intelligence.
